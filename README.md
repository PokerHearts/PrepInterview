# AI Interview Prep Tool

A free, open-source AI mock interview tool hosted as a static site. Practice for your next job interview with a realistic AI interviewer powered by Google Gemini.

## Features
- **BYOK (Bring Your Own Key):** Uses your own Gemini API key. No backend, no data storage.
- **Resume Parsing:** Upload your PDF resume to give the AI context.
- **Voice Interaction:** Real-time speech-to-text and text-to-speech with "Barge-in" support.
- **Automated Scorecard:** Get detailed feedback, strengths, and weaknesses after each session.
- **Privacy First:** All processing happens in your browser.

## How to Use
1. Clone this repository or download the files.
2. Open `index.html` in your browser or host it via GitHub Pages.
3. Enter your Gemini API Key.
4. Upload your resume and paste the job description.
5. Start practicing!

## Tech Stack
- Vanilla HTML/CSS/JS
- [Gemini API](https://ai.google.dev/)
- [PDF.js](https://mozilla.github.io/pdf.js/)
- Web Speech API & Web Audio API
